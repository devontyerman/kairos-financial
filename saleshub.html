How to Find Your Discord User ID

  1. Open Discord (desktop or mobile)
  2. Go to Settings (gear icon ⚙️)
  3. Scroll down to Advanced
  4. Toggle on Developer Mode
  5. Go back and tap/right-click your own name
  6. Select Copy User ID
  7. Paste it into the field — it'll look like 123456789012345678

You can always add this later from your profile settings.
